# Getting Started with Git

Before you can actually work with Git, you need to install it!

## Installing Git
- Download it online from [git-scm.com]

## Setting Up Your Identity
Get needs to know who you are so you can create repositories and make changes

- gitconfig --global user.name "Your Name"
- gitconfig --global user.email "your@email.com"

## Create a Repository
This command initializes a new repository in your project folder. You will have a hidden folder that stores your version history for your projects.

- git init
